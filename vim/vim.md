# vim

 * Moverte en el texto
Para moverte a la derecha izquierda arriba y abajo  (l  h  k j)
para moverte palabra por palabra (w)
para moverte atras de una palabra (b)
para ir al fina de una palabra (e)

Insertar texto
presiona (i) para insertar texto y esc dos veces para salir del modo insert
presiona (a) para insertar texto al final del cursor
presiona (A) para insertar texto al final de la linea

Eliminar texto 
presionar (x)  elimina el texto

Guardar Cambios
presiona (w) para guardar cambios
presiona (wq) para guardar y salir

Moverse entre archivos
ir a la definicion de una funcion   presiona (gd)
ir a la definicion en el archivo presion (gf)
mover entrea archivo (control + o) regresar (control + i )

Comandos para eliminar undo y redo
presion (dw) para eliminar palabras
para deshacer los cambios presiona (u) undo
para hacer rehacer  presion (control + r) redo 
para eliminar el todo el texto de una linea  (d + $) pero no elimina la linea

Eliminar la linea completamente presionar dos veces (d) 
para pegar una linea que fue removida presionando dos veces (d) presionar la (p) o (P) para agregar una nueva linea

Para remplazar un caracter de deber presionar la letra (r)

* Operador de Cambio
para cambiar una palabra se preiona (c+w) change word 
elminar una palabra para ser remplazada presionar (c+i+w)

* Saltos de  linea o cursor y buscar
ver linea presionar (control + g)
Para ir a la linea de inicio presionar dos veces (gg)
Para ir a la linea final presionar (G)
Para saltar ana linea debe presionar el numero de la linea y (G) ejemplo : (15 + G)

* Buscar
para buscar presionar (/) busca desde el cursor hacia abajo o (?) que busca desde el cursor hacia arriba
 luego escribir la  palabra a buscar para navegar entra las busqueda presiona (n) o (N)
